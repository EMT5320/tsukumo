# Tsukumo v0.1.0 — 可信交接与九十九工房 V0

Tsukumo v0.1.0 是作品集优先的首个可安装版本：它把 Claude、Codex 等外部 Agent 的事件归一为用户持有的 Chronicle、Canonical State 和 Handoff/Projection 记录，并通过 receipt-first 执行、只读 re-entry 与九十九工房 TUI 展示状态边界。

## Highlights

- Rust 五 crate 单向分层：Kernel / Adapters / Soul / Theater / Host；
- Claude / Codex fixture adapters 与 opt-in live seam；
- Chronicle、版本化 Canonical State、handoff 与不可变 `ProjectionReceipt`；
- `episode inspect` 对账 Git、artifact、open loop 和 runtime，显式区分 current/completed/drifted/blocked/unknown；
- `episode resume` 在启动已审阅 runtime 前提交 receipt，并保留失败语义；
- 九十九工房 HalfBlock TUI、Shiori 五态、State/Projection Inspector、Permission modal；
- Full / Compact / Fallback、truecolor / ANSI-256 / monochrome、reduced-motion；
- Linux 与 Windows GNU CI。

## Install

```bash
cargo install --git https://github.com/EMT5320/tsukumo \
  --tag v0.1.0 tsukumo-host --locked

tsukumo-host --version
```

预期：`tsukumo-host 0.1.0`。
